Tags/Elements

HTML "tags" and "elements" are basically the same thing. This very line is currently wrapped in a "paragraph" tag. Tags like this one have both an open and closed element, while tags like the "line-break" open and close themselves in one element.

HTML "skeleton"

We also learned about the standard parts of a usual html file. These include things like the !DOCTYPE statement, along with the head and body elements.

Common tags

We explored more common html tags such as p tags and the h1 through h6 tags (header).
